Notification guidelines

  • In-app — instant, no cost, delivered via SignalR when user is online.
  • Email — queued via MailKit/SMTP, arrives within minutes.
  • SMS — sent via Twilio, may incur costs. Use sparingly.
  • Keep messages concise and actionable.
  • Avoid sending more than 2–3 broadcast notifications per day.
